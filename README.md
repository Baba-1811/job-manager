# 就活管理アプリ

就職活動の企業管理・タスク管理・面接振り返りを一元管理するWebアプリです。

## デモ
https://xxxxx.vercel.app

## 機能

### 企業管理
- 企業登録
- 志望度管理
- 選考ステータス管理
- 編集 / 削除

### タスク管理
- ES締切管理
- 面接準備タスク
- 優先度表示
- 締切警告

### 面接振り返り
- 質問記録
- 自己評価
- 改善点
